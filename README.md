# Finance_Tracker

#### This is a simple Python terminal application designed to help users manage their personal finances. Users can record income and expenses, set monthly budgets, and view financial summaries to better understand spending habits. 

#### The application stores the financial data in a PostgreSQL database, the application gives the users easy options to view the data from the database.

## Features 

* Add income records 
* Add expense records 
* Remove income and exepense records
* Set monthly budgets 
* View spending by category
* View monthly financial summaries
* Store data in PostgreSQL Database
* Input validation and error handling

## Technologies used

* Python
* PostgreSQL
* psycopg2
* Object-Oriented Programming (OOP)

## Installation

1. Clone the repository
2. Open the project folder
3. Install the requirements 
`pip install psycopg2-binary`
4. Create a PostgreSQL database and the tables 'finace_tracker.sql'
5. Update the database connection details in 'config.py' and 'db_utils.py' (if needed)
6. Create & Activate the virtual enviroment 
`python -m venv .venv`
`source .venv/bin/activate`
6. Run the application
